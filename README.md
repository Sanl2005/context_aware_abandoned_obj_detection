\# Context-Aware Abandoned Object Intelligence System



This project monitors video feeds to detect unattended objects and estimate risk gradually

based on time, interaction, and context.



\## Tech Stack

\- Frontend: HTML, CSS, JavaScript

\- Backend: Ruby on Rails (API)

\- ML Service: Python Flask

\- DB: PostgreSQL, Redis



\## Folder Structure

\- ml\_service/ : Python Flask ML inference service

\- rails\_backend/ : Rails API backend

\- frontend\_dashboard/ : Web dashboard UI

\- docs/ : documents and notes
