\# Project Scope (Week 1)



\## Goal

Detect unattended objects (bags/parcels) in public spaces and evaluate risk gradually,

instead of triggering immediate panic alerts.



\## Modules

1\. Camera / Video input

2\. Object detection (YOLO)

3\. Multi-object tracking (DeepSORT)

4\. Interaction check (person near object or not)

5\. Time monitoring (idle time)

6\. Risk evaluation (low/medium/high)

7\. Rails backend storage + alerts

8\. Web dashboard for authorities



\## Output

\- Object list with ID, time, risk level

\- Alerts when risk crosses thresholds

\- History stored for analysis and audit
